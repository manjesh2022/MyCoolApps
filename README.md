# 🚀 My Cool Apps

A collection of useful web utilities, tools, and mini-applications built using HTML, CSS, and JavaScript.

## ✨ Features

- **Central Hub:** A clean homepage to easily navigate through all tools.
- **Interactive Tools:** Includes a Spinwheel, a Calculator, and a QR Code Generator.
- **Language Study:** Features a dedicated JLPT flashcard utility tool.
- **Billing & Management:** Quick-access tools like a Bill generator and presentation helper.
- **Fully Hosted:** Accessible from anywhere via GitHub Pages.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📦 What's Inside?

### 📁 Folders
- `0favicon/` - Contains the favicons and branding icons for the web pages.
- `MyClicks/` - A sub-project folder containing a responsive photography gallery.

### 📄 Core Web Pages
- `index.html` - The main homepage dashboard to access all utilities.
- `01_qlet.html` - A custom quizlet or flashcard utility tool.
- `02_blr.html` - A dedicated web page or layout tool.
- `03_bill.html` - A quick web-based bill or invoice manager.
- `04_spinwheel.html` - An interactive random picker wheel.
- `05_calci.html` - A fully functional web calculator.
- `06_qr.html` - A quick QR code generator.
- `07_ppt.html` - A custom web-based presentation helper.
- `08_jlpt.html` - A Japanese language study tool for JLPT preparation.
- `one.html` - An additional standalone web page utility.
